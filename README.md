# K 线训练系统 📈

基于 A 股真实**日 K 线**数据的专业训练系统，支持多种训练模式、技术指标分析、排行榜等功能。

## ✨ 功能特点

- 🎯 **短线训练** - 预测下一根 K 线涨跌方向（买入/卖出/持有/等待）
- 🔍 **短线盲测** - 隐藏股票代码，考验真实盘感
- 📈 **个股训练** - 针对特定股票深度训练
- 🔍 **个股盲测** - 个股训练盲测版本
- 🎯 **目标训练** - 设定收益目标，模拟真实交易
- 🎯 **目标盲测** - 目标训练盲测版本
- 📊 **技术指标** - MA(5/10/20/30)、MACD、KDJ、DMA（可切换）
- 🏆 **排行榜** - 按模式和K线数分类排名
- 📱 **响应式** - PC 端和手机端自适应显示
- 🎨 **专业配色** - A 股红涨绿跌标准配色
- ⚡ **本地 ECharts** - 图表库本地化，无需依赖 CDN

## 🚀 快速开始

### 1. 环境要求

- Python 3.8+
- macOS / Linux / Windows

### 2. 安装依赖

```bash
pip install baostock pandas numpy
```

> 💡 说明：本系统使用 Python 内置 HTTP 服务器，无需 Flask。

### 3. 获取数据

#### 方式一：使用 Qlib 离线数据（推荐）

如果你已经有 Qlib 数据，直接配置路径：

```bash
# Qlib 数据位置（默认）
~/.qlib/qlib_data/cn_data/stock/
├── sh/           # 上证股票
│   ├── 600519/   # 贵州茅台
│   │   └── 600519.csv
│   └── ...
├── sz/           # 深证股票
│   ├── 000858/   # 五粮液
│   │   └── 000858.csv
│   └── ...
└── bj/           # 北交所
    └── ...
```

> ⚠️ **重要提示**：本系统使用**日 K 线**数据，确保 CSV 文件包含 `date`（日期）字段。

#### 方式二：下载数据

```bash
# 下载 A 股日 K 线数据
python download_5min.py
```

### 4. 启动服务

```bash
python server.py
```

访问 http://localhost:8889

## 🎯 训练方法详解

### 一、训练模式（📊 训练）

**目标**：预测下一根 K 线的涨跌方向

**操作步骤**：

1. **选择 K 线数量**
   - 30 根：短线训练，考验即时判断
   - 60 根：标准训练（推荐）
   - 90 根：中线训练
   - 120 根：长线训练
   - 150 根：超长线训练

2. **设置每日目标**
   - 10 题：轻松模式
   - 20 题：标准模式（推荐）
   - 50 题：高强度训练

3. **可选：输入股票代码**
   - 上证：`sh.600519`（贵州茅台）
   - 深证：`sz.000858`（五粮液）
   - 科创：`sh.688981`
   - 北交：`bj.430047`
   - 留空：随机选择股票

4. **分析 K 线图**
   - 查看 K 线形态（阳线/阴线/十字星等）
   - 观察均线走势（MA5/MA10/MA20/MA30）
   - 分析 MACD 指标（DIF/DEA/柱线）
   - 参考 KDJ 或 DMA 指标（可切换）

5. **做出预测**
   - 📈 **买入**：预测下一根 K 线**收盘价 > 前一天收盘价**
   - 📉 **卖出**：预测下一根 K 线**收盘价 < 前一天收盘价**
   - ⏸️ **持有**：不确定，跳过（不计分）

6. **查看结果**
   - 系统揭晓下一根 K 线的实际走势
   - 显示预测是否正确
   - 更新得分和连胜记录

### 二、盲测模式（🔍 盲测）

**目标**：在不知道股票代码的情况下，纯靠技术分析预测

**与训练模式的区别**：
- 股票代码显示为 `??????`
- 无法通过个股特征（如茅台的稳定性）来判断
- 更能考验真实的技术分析能力
- 训练结束后揭晓股票代码

**适合人群**：
- 有一定技术分析基础的投资者
- 想要客观评估自己判断能力的人

### 三、个股训练（📈 个股训练）

**目标**：针对特定股票进行深度训练

**操作步骤**：
1. 输入股票代码（如 `sh.600519`）
2. 系统按时间顺序展示该股票的 K 线
3. 每次预测后，自动加载下一段 K 线
4. 可以观察同一股票在不同阶段的走势规律

**优势**：
- 熟悉特定股票的波动特征
- 学习识别该股票的支撑位和压力位
- 建立对个股的直觉判断

### 四、个股盲测（🔍 个股盲测）

**目标**：个股训练的盲测版本

**特点**：
- 隐藏股票代码
- 按时间顺序展示
- 训练结束后揭晓股票代码

### 五、排行榜（🏆 排行榜）

**排名维度**：
- 📅 **日榜**：当天成绩排名
- 📆 **周榜**：本周累计成绩
- 🗓️ **月榜**：本月累计成绩
- 📋 **总榜**：历史总成绩
- 🔍 **盲测榜**：仅盲测模式成绩

**评分规则**：
- 基础分：预测正确 +10 分
- 连胜加成：连续正确额外 +2/+4/+6/... 分
- 连胜中断：重置为 0
- 持有观望：不计分

## 📊 技术指标说明

### 副指标切换

页面顶部有 **KDJ** 和 **DMA** 两个按钮，点击可切换下方副指标区域的曲线显示。

### KDJ 指标

| 线 | 含义 | 参考值 |
|----|------|--------|
| K 线 | 快速随机指标 | 20 以下超卖，80 以上超买 |
| D 线 | 慢速随机指标 | 与 K 线交叉产生信号 |
| J 线 | K、D 的偏离程度 | 可能 >100 或 <0 |

**使用技巧**：
- K 线上穿 D 线（金叉）：买入信号
- K 线下穿 D 线（死叉）：卖出信号
- J 线 >100：超买，注意回调
- J 线 <0：超卖，注意反弹

### DMA 指标

| 线 | 含义 | 参考值 |
|----|------|--------|
| DMA 线 | 短期均线 - 长期均线 | 反映趋势强弱 |
| AMA 线 | DMA 的移动平均 | 与 DMA 交叉产生信号 |

**使用技巧**：
- DMA 上穿 AMA：买入信号
- DMA 下穿 AMA：卖出信号
- DMA >0 且上升：多头趋势
- DMA <0 且下降：空头趋势

### MACD 指标

| 线 | 含义 | 参考值 |
|----|------|--------|
| DIF 线 | 快速 EMA - 慢速 EMA | 反映短期趋势 |
| DEA 线 | DIF 的移动平均 | 与 DIF 交叉产生信号 |
| MACD 柱 | (DIF - DEA) × 2 | 红涨绿跌 |

**使用技巧**：
- DIF 上穿 DEA（金叉）：买入信号
- DIF 下穿 DEA（死叉）：卖出信号
- 红柱变长：多头增强
- 绿柱变长：空头增强

### 均线系统

| 线 | 含义 | 参考值 |
|----|------|--------|
| MA5 | 5 日均线 | 短线趋势 |
| MA10 | 10 日均线 | 中短线趋势 |
| MA20 | 20 日均线 | 中线趋势 |
| MA30 | 30 日均线 | 中长线趋势 |

**使用技巧**：
- 均线多头排列（MA5>MA10>MA20>MA30）：强势上涨
- 均线空头排列（MA5<MA10<MA20<MA30）：弱势下跌
- 均线交叉：趋势转换信号

## 💰 买入持有基准

**含义**：如果在 K 线图的第一天买入，一直持有到最后一天，收益率是多少？

**计算公式**：
```
买入持有收益率 = (最后一天收盘价 - 第一天收盘价) / 第一天收盘价 × 100%
```

**用途**：
- 对比你的预测准确率
- 评估频繁操作是否优于长期持有
- 如果你的预测收益跑不赢"买入持有"，说明操作反而不如躺平 😄

## 📊 数据格式

### Qlib CSV 格式（**日 K 线**）

```csv
date,open,high,low,close,volume,amount,adjustflag,factor
2024-01-02,1800.01,1810.50,1795.00,1805.30,12345678,22345678901,3,1.0
2024-01-03,1810.00,1815.00,1805.00,1812.50,10234567,20345678901,3,1.0
```

### 字段说明

| 字段 | 说明 |
|------|------|
| date | **日期**（YYYY-MM-DD 格式） |
| open | 开盘价 |
| high | 最高价 |
| low | 最低价 |
| close | 收盘价 |
| volume | 成交量 |
| amount | 成交额 |
| adjustflag | 复权标志 (1:后复权 2:前复权 3:不复权) |
| factor | 复权因子 |

## 🔧 配置说明

编辑 `config.json` 文件：

```json
{
  "data_dirs": [
    "~/.qlib/qlib_data/cn_data/stock",
    "data/daily"
  ],
  "port": 8889
}
```

### 数据目录

`data_dirs` 支持多个数据目录，自动识别两种格式：

| 格式 | 目录结构 | 说明 |
|------|---------|------|
| Qlib | `{dir}/{sh\|sz}/{code}/{code}.csv` | Qlib 离线数据 |
| baostock | `{dir}/{code}.csv` | 脚本下载的日线数据 |

**添加新数据源**：只需在 `data_dirs` 中加入路径，重启服务即可。

### 修改端口

修改 `config.json` 中的 `port` 字段。

## 📱 使用技巧

### 1. 结合多个指标

不要只看单一指标，结合 K 线形态、均线、MACD、KDJ/DMA 综合判断。

### 2. 注意成交量

- 放量上涨：多头强势，可能继续上涨
- 放量下跌：空头强势，可能继续下跌
- 缩量上涨：上涨动力不足
- 缩量下跌：下跌动力不足

### 3. 观察趋势

- 上升趋势中：回调买入
- 下降趋势中：反弹卖出
- 震荡行情：观望为主

### 4. 设定目标

每天完成一定数量的训练（如 20 题），持续提升盘感。

### 5. 记录错误

记录自己预测错误的情况，分析错误原因，避免重复犯错。
